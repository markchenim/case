# 相关司法解释与指导意见

> 最高人民法院、最高人民检察院及地方司法机关的相关文件

## 一、最高人民法院相关司法观点

### 1.1 代孕协议效力的司法态度

**最高人民法院《全国法院民商事审判工作会议纪要》（九民纪要，2019 年）**

九民纪要第 31 条明确：

> 违反规章一般情况下不影响合同效力，但该规章的内容涉及**金融安全、市场秩序、国家宏观政策**等公序良俗的，应当认定合同无效。

> 📌 代孕协议虽未被民法典明确列为无效合同类型，但因涉及**人类伦理、生育秩序等公序良俗**，法院普遍认定其无效。

### 1.2 最高人民法院的长期立场

最高院在多个审判工作座谈及指导意见中强调：

- 代孕涉及社会伦理和人类尊严
- 代孕协议因违背公序良俗而无效
- 代孕产生的亲子关系认定应坚持"儿童利益最大化"

## 二、上海高级人民法院《关于审理代孕纠纷案件的指导意见》

> **全国首个专门针对代孕纠纷的地方性司法指导意见**

### 2.1 主要内容

| 要点 | 内容 |
|------|------|
| 协议效力 | 代孕协议因违反公序良俗，应认定无效 |
| 亲子关系认定 | 坚持"分娩说"——代孕母为法律母亲 |
| 儿童利益 | 在处理子女抚养问题时以儿童利益最大化为核心 |
| 费用返还 | 已支付的代孕费用，一般不予返还 |
| 损害赔偿 | 代孕导致人身损害的，可依据侵权责任法主张赔偿 |

### 2.2 亲子关系的认定规则

| 情形 | 法律父母认定 |
|------|-------------|
| 代孕母使用自己的卵子（传统代孕） | 代孕母为母亲 |
| 代孕母使用委托方的受精卵（妊娠代孕） | 委托方基因母亲可主张确认亲子关系 |
| 委托方与代孕母之间有协议 | 协议无效，但不影响亲子关系的事实认定 |
| 委托方已实际抚养子女 | 考虑儿童利益，可认定为事实上的抚养关系 |

### 2.3 抚养权争议的处理原则

1. 以子女利益最大化为首要原则
2. 考虑子女实际生活状况和情感联系
3. 委托方与代孕母均有主张抚养权的权利
4. 哺乳期内的子女原则上随母生活

## 三、最高人民法院相关指导案例

### （2015）沪一中少民终字第 56 号

> 上海市第一中级人民法院，载于最高人民法院《人民法院案例选》

**案件要点：**

- 夫妻委托代孕生下龙凤胎
- 丈夫去世后，公婆与儿媳争夺子女抚养权/监护权
- 法院认定：代孕所生子女的亲子关系认定，应以**儿童最大利益**为原则
- 法院确认：**分娩者（代孕母）为法律母亲**；委托方可通过**收养**或**继父母子女关系**建立法律联系

**影响：**
本案是代孕亲子关系认定的标志性案件，被最高院收录为指导性案例参考。

## 四、最高人民检察院相关意见

### 4.1 公益诉讼

最高检在代孕领域的司法态度：

- 非法代孕侵害女性健康权、身体权
- 代孕中介涉嫌非法经营、非法行医
- 检察机关可提起公益诉讼（涉及公共卫生、妇女权益）

### 4.2 典型案例（2022 年）

最高检发布的一起公益诉讼指导案中：
- 某代孕中介机构被判处赔偿公益损失
- 涉案医护人员被追究刑事责任
- 代孕母获得医疗救助和心理辅导

| 地区 | 司法倾向 |
|------|----------|
| 上海 | 指导意见最明确，体系最完善 |

## 六、亲子关系认定的法律路径

| 路径 | 可行性 | 说明 |
|------|--------|------|
| 分娩说（代孕母为母） | ✅ | 司法实践的普遍做法 |
| 基因说（提供卵子者为母） | ⚠️ | 需个案评估，非通说 |
| 收养 | ⚠️ | 需满足收养法条件 |
| 继父母子女关系 | ⚠️ | 限于委托方与代孕母有婚姻关系 |
| 事实抚养关系 | ✅ | 法院在抚养权案件中会考虑 |
