# 代孕相关法律以及实操知识库

> 系统性整理中国代孕（Surrogacy）相关法律法规、政策文件、实务操作指引、典型案例与参考资料。

---

## 📋 项目说明

代孕在中国是一个**法律全面禁止但实践中仍存在且争议巨大**的领域。2021 年《人口与计划生育法》修正案首次在法律层面明确 "禁止以任何形式实施代孕"。然而，跨境代孕、代孕纠纷、代孕相关刑事犯罪等问题持续引发社会关注。

本知识库以现行法律框架为基础，从法律条文、司法实践、实务操作、案例研究四个维度，系统梳理代孕相关的法律问题和应对策略，旨在为法律从业者、研究者和有需要的社会公众提供全面、客观的参考。

> ⚠️ **核心提示**：代孕在中国属于违法行为。本知识库仅供法律研究、风险教育与实务参考，不鼓励或指导实施代孕行为。
