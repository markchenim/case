# 意定监护法律与实操知识库

> 系统性整理中国意定监护（Voluntary Guardianship / Adult Guardianship）相关法律法规、实务操作流程、典型案例与参考资料。

---

## 📋 项目说明

意定监护，是指成年人在具备完全民事行为能力时，通过书面协议预先指定自己信任的人或组织作为监护人，在自己未来丧失或部分丧失民事行为能力时，由该监护人代为处理人身、财产等相关事务的法律制度。

本知识库以 **《民法典》第 33 条** 为核心，系统梳理了意定监护的法律基础、实操流程、公证指引、案例参考和扩展阅读资料，旨在为法律从业者、有需求的社会公众以及相关机构提供一站式参考。
