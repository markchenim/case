# 老年人权益保障法·意定监护相关条文

> 《中华人民共和国老年人权益保障法》（2018 年修正）

---

## 一、第二十六条 —— 老年人意定监护

> **具备完全民事行为能力的老年人，可以在近亲属或者其他与自己关系密切、愿意承担监护责任的个人、组织中协商确定自己的监护人。监护人在老年人丧失或者部分丧失民事行为能力时，依法承担监护责任。**
>
> **老年人未事先确定监护人的，其丧失或者部分丧失民事行为能力时，依照有关法律的规定确定监护人。**

### 条文解读

| 要点 | 说明 |
|------|------|
| 适用范围 | 年满 60 周岁的老年人 |
| 能力要求 | 具备完全民事行为能力 |
| 监护人来源 | 近亲属、关系密切的个人或组织 |
| 形式要求 | 协商确定（建议结合民法典第 33 条以书面形式） |
| 生效条件 | 老年人丧失或部分丧失民事行为能力时 |
| 法律衔接 | 与《民法典》第 33 条意定监护制度一致，老年人权益保障法是最早引入意定监护理念的法律之一 |

## 二、其他相关条款

### 第二十四条 —— 养老服务的自主选择

> 鼓励医疗机构等为老年人提供疾病预防、医疗、护理、康复、心理咨询等服务。
>
> 提倡为老年人建立健康档案，定期体检。

> 📌 意定监护实践中，可结合医疗服务委托代理（医疗预嘱）一并安排。

### 第二十五条 —— 禁止侵害老年人权益

> 禁止对老年人实施家庭暴力。
>
> 禁止家庭成员歧视、侮辱、诽谤、遗弃老年人。

### 第二十七条 —— 国家建立健全养老服务体系

> 国家建立健全居家为基础、社区为依托、机构为支撑的社会养老服务体系。
>
> 倡导全社会优待老年人。

## 三、意定监护与《老年人权益保障法》的衔接要点

```
《民法典》第 33 条          ← 一般法（通用规则）
        ↓
《老年人权益保障法》第 26 条  ← 特别法（对老年人的专门保护）
```

> 🧓 **特别法与一般法的关系**：对老年人意定监护，优先适用《老年人权益保障法》第 26 条；该条未规定的事项，适用《民法典》总则编关于监护的一般规定。

---

## 四、老年人的特殊保护原则

| 原则 | 内容 |
|------|------|
| 尊重意愿 | 监护安排应充分尊重老年人的真实意愿 |
| 有利原则 | 处理涉老事务应最有利于老年人 |
| 国家责任 | 国家通过立法、行政、司法多重手段保护老年人权益 |
| 社会参与 | 鼓励社会组织、志愿者参与养老服务与监护 |
