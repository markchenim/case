## 一、司法案例

### 案例 1：上海市首例意定监护确认案

**案号**：(2018) 沪 01XX 民特 XX 号

**基本案情：**
- 委托人王某某（女，时年 78 岁），独居，无子女，配偶已故
- 与侄女李某关系亲密，希望通过意定监护指定李某为监护人
- 已办理意定监护协议公证
- 后因患阿尔茨海默病，医院出具诊断证明
- 侄女李某申请确认监护人身份

**裁判结果：**
法院依照《民法典》第 33 条、《老年人权益保障法》第 26 条，确认李某为王某某的意定监护人。

**裁判要点：**
1. 经公证的意定监护协议效力优先于法定监护顺序
2. 公证文书具有高度证明力
3. 尊重委托人（被监护人）的真实意愿

**典型意义：**
上海首例法院确认意定监护人的案件，为意定监护制度的司法适用提供了先例。

### 案例 2：意定监护与法定监护的冲突案

**基本案情：**
- 委托人赵某（男，时年 82 岁），有两子一女
- 赵某与女儿关系不睦，指定儿子赵某某为意定监护人
- 后赵某患脑梗，意识不清
- 女儿主张自己按法定顺序应为监护人之一，要求参与监护事务

**争议焦点：**
意定监护协议能否排除法定监护人的监护权？

**裁判结果：**
法院确认意定监护协议有效，赵某某为唯一监护人。女儿不服上诉，二审维持原判。

**裁判要点：**
1. 意定监护优先于法定监护
2. 法定监护人不得以"法定顺序"为由对抗经公证的意定监护协议
3. 但法院同时要求赵某某每半年向其他子女通报情况

**典型意义：**
明确了意定监护对法定监护的优先效力，但也提示了家庭矛盾需要妥善处理。

### 案例 3：监护人不当履职被撤销案

**基本案情：**
- 委托人孙某指定朋友张某为意定监护人
- 张某在孙某入住养老院后，擅自将孙某的房产以低于市场价 30% 的价格出售
- 购房款去向不明
- 孙某的侄子向法院申请撤销张某的监护人资格

**裁判结果：**
法院撤销张某的监护人资格，并责令张某赔偿孙某的财产损失。同时指定孙某的侄子为监护人（原意定监护协议已约定替补监护人）。

**裁判要点：**
1. 监护人不得为自身利益处分被监护人财产
2. 违反监护职责的，依法承担赔偿责任
3. 设置替补监护人的价值在此案中充分体现

**典型意义：**
警示意定监护中监督机制的重要性，建议在协议中加入监督条款和财产处分限制。

## 二、公证实务案例

### 案例 4：同性伴侣的意定监护安排

**背景：**
- 委托人钱先生（男，45 岁）与伴侣周先生（男，43 岁）同居 15 年
- 钱先生希望患病时由周先生代为做医疗决策
- 因双方非法律承认的配偶关系，医院不认可周先生的身份

**方案：**
- 钱先生与周先生签署意定监护协议并公证
- 指定周先生为医疗决策代理人
- 同时设定监督人（钱先生的妹妹）
- 将公证书副本交至常就诊医院备案

**结果：**
- 公证书得到医院认可
- 周先生能够代为签署医疗同意书
- 钱先生说"这张公证书比结婚证还管用"

**典型意义：**
意定监护为法律不能认可的家庭关系提供了替代性保护方案。

### 案例 5："遗嘱+意定监护+信托"综合方案

**背景：**
- 委托人陈女士（时年 70 岁），丧偶，独生子有智力障碍
- 担忧自己去世后儿子无人照顾、财产被侵占

**方案：**
- **意定监护协议**：指定侄女在陈女士丧失行为能力时担任监护人，照顾陈女士及其子
- **遗嘱**：将遗产设立信托
- **信托**：以陈女士之子为受益人，侄女为信托监察人

**结果：**
- 形成"人+财"双重保障体系
- 陈女士丧失行为能力后，侄女管理财产、照顾母子
- 陈女士去世后，信托继续为儿子提供生活保障

**典型意义：**
意定监护+遗嘱+信托组合使用，为特殊需要家庭成员提供全方位的保障。

---

### 案例 6：失独老人的多元化选择

**背景：**
- 刘叔叔和周阿姨，均 75 岁，独生子女已故
- 无可靠亲属可担任监护人
- 担忧未来无人照料

**方案：**
- 选择某社会组织担任意定监护人（该组织为民政部门批准的专业监护机构）
- 同时指定社区居委会为监督人
- 将房产捐赠予该组织，作为对价，该组织终身照顾两位老人

**结果：**
- 该组织履行监护职责，安排养老、医疗事宜
- 社区居委会定期回访
- 老人去世后，房产按意愿处理

**典型意义：**
组织/机构也可以担任意定监护人，为无亲属可托付的人群提供了新的选择。

## 三、实务经验总结

### 成功案例的共同特征

| 特征 | 说明 |
|------|------|
| 及早规划 | 在身体健康、意识清晰时办理 |
| 公证加持 | 全部经过公证，执行力强 |
| 范围明确 | 监护范围具体，没有模糊地带 |
| 监督到位 | 设置了监督人或报告机制 |
| 家属知情 | 重要亲属知晓安排，减少纠纷 |
| 备用方案 | 有替补监护人/phased arrangement |

### 失败案例的教训

| 教训 | 反面案例摘录 |
|------|-------------|
| 不公证 | 未经公证，医院不认可，导致无法手术 |
| 范围模糊 | "一切事务"过于笼统，监护人不知从何下手 |
| 无监督 | 监护人挪用财产，因无监督时隔多年才发现 |
| 亲属不知情 | 子女否认协议效力，诉至法院耗时一年 |
| 未告知机构 | 银行拒绝监护人取款，因无备案记录 |
