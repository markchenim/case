# 意定监护协议·核心要点与条款设计

> 本文件提供意定监护协议的条款设计与注意事项，供起草协议时参考。

---

## 一、协议的核心要素

根据《民法典》第 33 条，一份有效的意定监护协议必须包含：

```
┌────────────────────────────────────────────────┐
│                意定监护协议                       │
├────────────────────────────────────────────────┤
│  1. 委托人信息（完全民事行为能力人）              │
│  2. 监护人信息                                  │
│  3. 协商一致的书面记载                           │
│  4. 明确的监护职责范围                           │
│  5. 生效条件（丧失/部分丧失行为能力时）          │
│  6. 双方签字 + 日期                             │
└────────────────────────────────────────────────┘
```

---

## 二、条款设计详解

### 2.1 委托人与监护人信息

| 字段 | 说明 |
|------|------|
| 姓名 | 与身份证一致 |
| 公民身份号码 | 18 位 |
| 住所地 | 实际居住地址 |
| 联系方式 | 电话、微信、邮箱等 |
| 紧急联系人 | 第三方可联系的人 |

### 2.2 监护范围条款（核心条款）

**监护范围可选择包含：**

**① 人身照管事务**

```
□ 医疗决策权
   └─ 选择医疗机构
   └─ 同意/拒绝治疗方案（包括手术、化疗等）
   └─ 临终医疗决策（是否延续生命）
   └─ 遗体处理与丧葬安排
□ 生活照料安排
   └─ 选择养老机构/护理机构
   └─ 聘用护理人员
   └─ 安排日常起居
□ 居住安排
   └─ 决定居住地点（居家/养老院/护理院）
   └─ 代表签订养老服务合同
□ 其他：___________________________
```

**② 财产管理事务**

```
□ 日常财务管理
   └─ 支取银行存款用于生活开支
   └─ 领取退休金/养老金/补贴
   └─ 支付医疗费、护理费、住房费等
□ 重大财产处分（建议设置额外限制）
   └─ 出售/出租/抵押房产（需监督人同意）
   └─ 大额投资（超过___元需监督人同意）
   └─ 赠与（建议禁止或限制）
□ 税务与保险
   └─ 代为申报纳税
   └─ 办理保险理赔
□ 其他：___________________________
```

**③ 法律事务代理**

```
□ 代为提起诉讼/应诉
□ 委托律师
□ 签署法律文件
□ 处理行政事务（户籍、社保等）
□ 其他：___________________________
```

### 2.3 限制与排除条款

> ❗ **重要：** 明确列出监护人不可以做什么

**建议限制条款：**

```
禁止事项（监护人不得）：
□ 不得处分委托人的不动产，除非为支付医疗费且经监督人书面同意
□ 不得以委托人财产为监护人自己或第三人提供担保
□ 不得将委托人财产转移至监护人名下
□ 不得代理委托人订立遗嘱
□ 不得代理委托人结婚、离婚
□ 不得进行大额捐赠（小额公益捐赠除外）
□ 不得代理委托人签订遗赠扶养协议
□ 其他：___________________________
```

### 2.4 监护监督条款

**监督人的设置：**

```
监督人信息：
  姓名：________  身份证号：________________
  联系方式：________  与委托人关系：________

监督人职权：
□ 定期（每___个月/年）接收监护人履职报告
□ 审查财产管理报告
□ 对重大财产处分有否决权
□ 发现监护人不当履职时向法院申请撤销监护人资格
□ 在监护人缺位时担任临时监护人（如约定）
```

### 2.5 替补监护人条款

> 建议设置至少一名替补监护人，以防主监护人先于委托人去世或丧失履职能力。

```
替补监护人（顺位）：
  第一替补：________  身份证号：________________
  第二替补：________  身份证号：________________

替补监护人履职条件：
  主监护人去世/丧失行为能力/辞任/被撤销资格时自动替补
```

### 2.6 生效条款

```
生效条件：
本协议自委托人丧失或部分丧失民事行为能力之日起生效。

行为能力丧失的认定方式（选择一种或多种）：
□ 二级以上医疗机构出具的诊断证明
□ 司法鉴定机构出具的行为能力鉴定意见
□ 人民法院宣告为限制/无民事行为能力人
□ 经监督人与监护人共同确认

过渡安排：
在正式生效认定作出前，如委托人处于暂时无法表达意愿的紧急状态，
监护人可凭本公证书先行代为做出医疗决策。
```

### 2.7 终止条款

```
本协议在以下情形终止：
① 委托人恢复完全民事行为能力
② 委托人去世
③ 委托人单方撤销协议（以书面形式送达监护人）
④ 双方协商一致解除协议
⑤ 监护人去世/丧失行为能力/被撤销监护人资格且无替补监护人
⑥ 人民法院撤销监护人资格
```

### 2.8 报酬条款

```
监护人报酬：
□ 无偿监护
□ 有偿监护：每月_____元
□ 实报实销：因履行监护职责产生的合理费用由委托人承担
□ 混合方式：___________________________
```

## 三、特殊场景的条款设计

### 3.1 医疗预嘱条款

> 将生前预嘱（Living Will）与意定监护结合

```
医疗意愿声明：
  本人希望______：
  □ 尽量延长生命，使用一切可能的医疗手段
  □ 在不可逆转的末期病情下，不采取过度治疗
  □ 安宁疗护（姑息治疗）
  □ 器官捐献意愿：________
  □ 遗体处理方式：________
```

### 3.2 财产管理方案

```
财产管理原则：
  以维持委托人生活、医疗、护理需求为首要目标
  投资应以保本稳健为原则
  每年财产管理报告应经监督人审核
```

### 3.3 紧急联络人

```
如监护人无法及时履职，下列人员可作为紧急联络人：
  姓名：________  关系：________  电话：________
  姓名：________  关系：________  电话：________
```

## 四、协议的签署与保管

### 签署要求

```
签署人：
  委托人：________（手写签名）
  监护人：________（手写签名）
  监督人（如有）：________（手写签名）

签署日期：____年__月__日
签署地点：________
```

### 保管方式

```
正本保管：
  1. 交公证处存档（如公证）
  2. 委托人保管原件（建议保险箱）
  3. 监护人保管副本
  4. 可委托律师事务所保管

备份建议：
  □ 银行保管箱
  □ 公证云平台电子备份
  □ 告知一位可靠亲属存放位置
```

## 五、律师审查清单

- [ ] 委托人是否具备完全民事行为能力？
- [ ] 委托人的意思表示是否真实、自愿？
- [ ] 监护人是否具备完全民事行为能力？
- [ ] 协议内容是否违反法律强制性规定？
- [ ] 监护范围是否明确、具体？
- [ ] 禁止/限制条款是否清晰？
- [ ] 生效条件是否明确可操作？
- [ ] 是否有终止条款？
- [ ] 是否设置了必要的监督机制？
- [ ] 涉及重大财产处分的，是否有额外保护？
- [ ] 是否需要设置替补监护人？
- [ ] 是否考虑过监护人的报酬/费用承担？
